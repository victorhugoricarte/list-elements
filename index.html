✏️ Tipos de listas

1. Lista ordenada:

</ol> ordered list / </li> list item

→ Exemplo 1 (normal):

    Lista de tarefas:

  1. Escrever diário de bordo
  2. Fazer a primeira refeição do dia
  3. Voltar ao diário de bordo

→ Exemplo 2 com atributos start e type → parâmetros (1, A, a, I, i):

    Pedaço da mesma lista de tarefas [Noite]:

  1. Fazer manutenção no repositório
  2. Realizar ajustes do piloto-seguro
    1. Revisar cabeamento
    2. Revisar as placas
    3. Fechar com parafusos G-LM
  3. Fazer a quinta refeição do dia

2. Lista não-ordenada:

</ul> unordered list / </li> list item

→ Exemplo com atributo type → parâmetros (disc > circle > square):

  • Fazer café
  • Estudar
    • Matemática: Geometria
      • Retângulos
      • Círculos
    • Biologia
    • IHC
  • Tomar café da manhã
  • Almoçar

3. Lista de definição:

</dl> definition list / </dt> definition term → </dd> definition details

Uma lista de definição é desenvolvida dentro de um elemento </dl>, definindo dentro dela o termo a ser definido </dt> e a definição em si </dd>.

→ Estrutura:

Início da lista de definicação (definition list)

Termo de Definição especificado (definition term)
Os detalhes dessa definição (definition details)

Fim da lista de definicação (definition list)

→ Exemplos:

HTML
Uma linguagem de marcação de hipertexto, cria o conteúdo de um site.
CSS
Uma linguagem de estilização de folhas em cascata, cria o design de um site.
JavaScript
Uma linguagem de programação, cria a interatividade de um site.
